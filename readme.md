# azure-devops-extension-mock

This project is a mock implementation of the [Azure DevOps Extension SDK](https://www.npmjs.com/package/azure-devops-extension-sdk) and [Azure DevOps Extension API](https://www.npmjs.com/package/azure-devops-extension-api). It is intended to allow developers to test their extensions without having to deploy them to an actual Azure DevOps instance.

## Installation

To install the package, simply run: `npm install --save-dev azure-devops-extension-mock`
